# Claude Code Built-in Commands

## Session
- `/clear` / `/new` — Clear history and free context
- `/resume` — Resume a past conversation
- `/rewind` — Rewind to a prior point
- `/rename` — Rename the current session
- `/compact` — Compress conversation history

## Model & Settings
- `/model` — Switch AI models
- `/effort` — Set effort level (low/medium/high/max)
- `/fast` — Toggle fast mode
- `/config` / `/settings` — Open settings UI
- `/theme` — Change color theme

## Context & Info
- `/context` — Visualize context usage
- `/cost` — Show token usage stats
- `/status` — Show version, model, connectivity

## Tools & Permissions
- `/permissions` — Manage tool permissions
- `/add-dir` — Add a working directory
- `/mcp` — Manage MCP server connections
- `/hooks` — View hook configs

## Skills (Prompt Workflows)
- `/skills` — List available skills
- `/simplify` — Review changed code for quality
- `/loop` — Run prompts on a recurring interval
- `/plan` — Enter plan/architect mode

## Code & Project
- `/init` — Initialize project with CLAUDE.md
- `/diff` — View uncommitted changes
- `/security-review` — Analyze changes for vulnerabilities

## Memory
- `/memory` — Edit CLAUDE.md and manage auto-memory

## Account
- `/login` / `/logout`
- `/usage` — Show plan usage and rate limits

## Utilities
- `/help` — Show help
- `/doctor` — Diagnose installation issues
- `/exit` / `/quit` — Exit Claude Code
